Class Overview

HSBeacon is used to display the Beacon SDK as well as handle primary functionality of the SDK, such as push notification handling and shortcut processing.

HSBeaconUser allows you to authenticate as a specific user, and attach custom attributes to help identify customers.

HSBeaconSettings contains configuration details for Beacon, such as your Beacon ID.

HSBeaconMessagingSettings provides configuration settings for the messaging features of Beacon.

HSBeaconContactForm allows you to provide pre-filled contact information for users, saving them time when contacting support.
